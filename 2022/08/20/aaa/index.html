frida 从入门到入门

安装

pip install frida-tools 就是这么简单!

注入方式

  • 直接注入
    • frida -l xx.js com.xxx
    • frida -l xx.js com.xxx -F 注入当前应用
  • 启动时注入(Spawn)
    • frida -l xx.js -f com.xxx --no=pause 不加--no=pause还得在交互界面补一句%resume(真的麻烦＞﹏＜)

食用教程

跨平台真的是太方便了，大大降低了学习成本，好耶！！！

Windows

函数拦截器（Interceptor）

Interceptor.attach(函数指针，回调函数)
回调函数：

  • onEnter：给定参数用于读写它们
  • onLeave：可以省略，给定返回参数，用于读写返回值（不同的返回值类型，需要相应的转换！）.replace()用来改写返回值
  • 注意：retval 对象会在 onLeave 调用中回收, 因此不要将其存储在回调之外使用, 如果需要存储包含的值, 需要制作深拷贝, 如 ptr(retval.toString())

主动调用

new NativeFunction(address, returnType, argTypes[, options])

  • address : 函数地址
  • returnType : 指定返回类型
  • argTypes : 数组指定参数类型
  • 类型可选: void, pointer, int, uint, long, ulong, char, uchar, float, double, int8, uint8, int16, int32, uint32, int64, uint64; 参照函数所需的 type 来定义即可;

内存操作

Memory.scan(地址，大小，模式，回调函数)

  • 模式：可以使用以？来表示任意数值的通配符
  • 回调函数：
    • onMatch(address,size):扫描到一个内存块，起始地址是address，大小size的内存块，返回stop表示停止扫描
    • onError:(reason):扫描内存的时候出现内存访问异常的时候回调
    • onComplete():内存扫描完毕的时候调用
